────────────────────────────────────────────────────────────────────────────────
  VUE D'ENSEMBLE
────────────────────────────────────────────────────────────────────────────────

Le projet Axion Pad se compose de deux parties distinctes :

  1. APPLICATION DE CONFIGURATION (Java / JavaFX)
     → Logiciel desktop pour configurer le clavier macro Axion Pad
     → Dossier : src/

  2. SITE E-COMMERCE (Node.js / HTML-CSS-JS)
     → Site marchand complet pour vendre l'Axion Pad
     → Dossier : website/


================================================================================
  PARTIE 1 — APPLICATION JAVA (Axion Pad Configurator)
================================================================================

  DESCRIPTION
  ───────────
  Application desktop JavaFX permettant de configurer le clavier macro Axion Pad
  (microcontrôleur RP2040 + firmware CircuitPython). L'utilisateur peut assigner
  des actions à chacune des 12 touches et 4 potentiomètres, puis exporter la
  configuration vers le périphérique.

  STACK TECHNIQUE
  ───────────────
  • Java 17
  • JavaFX 21
  • Maven (build & packaging)
  • jSerialComm 2.11.0 (communication série USB)
  • Gson 2.10.1 (persistance JSON)
  • JNA (Java Native Access — hook clavier Windows & API audio WASAPI)

  COMPILATION & LANCEMENT
  ────────────────────────
  Développement   : mvn javafx:run
  Build JAR       : mvn clean package
  Build .exe      : scripts\build-windows.bat
  Livrable .exe   : dist\windows\AxionPadConfigurator\AxionPadConfigurator.exe
  Archive ZIP     : dist\windows\AxionPadConfigurator-1.0.0-windows.zip

  ARCHITECTURE — PATTERN MVC
  ───────────────────────────
  • Pas de fichiers FXML — toute l'UI est construite programmatiquement
  • MainWindow gère le routage de pages via showPage(id)
  • Pattern singleton pour ConfigService, SettingsService, SerialService,
    WindowsVolumeService, KeyHookService

  FLUX DE DONNÉES
  ───────────────
  Série → UI :
    SerialService.readLoop() [thread BG]
    → parse "v1|v2|v3|v4\n"
    → Platform.runLater() → SoundbarController.updateSliderValues()
    → WindowsVolumeService.applySliderValues() [thread COM]

  Touches → Actions :
    RP2040 → USB HID → KeyHookService (WH_KEYBOARD_LL)
    → F13-F24 interceptés → executeAction(KeyConfig)
    → KEYBOARD : Robot.keyPress/keyRelease
    → APP      : ProcessBuilder cmd /c start
    → MUTE     : WindowsVolumeService.toggleMute(target)
    → MEDIA    : User32.SendInput (VK_VOLUME_*, VK_MEDIA_*)
    → AHK      : ProcessBuilder cmd /c start script.ahk

  Config → Firmware :
    PadConfig.generateCodePy() → ConfigService.exportCodePy()
    → copie sur lecteur CIRCUITPY (auto-détection A:-Z:)
    → CircuitPython redémarre automatiquement

================================================================================
  PARTIE 2 — SITE E-COMMERCE (Axion Pad Store)
================================================================================

  DESCRIPTION
  ───────────
  Site marchand complet et autonome pour vendre l'Axion Pad en ligne.
  Inclut catalogue produits, panier, tunnel d'achat, comptes clients,
  paiement Stripe, suivi de commandes et back-office admin.

  STACK TECHNIQUE
  ───────────────
  Backend  : Node.js 22 + Express 4
  Base de données : SQLite via better-sqlite3 v11 (synchrone, sans ORM)
  Auth     : JWT (jsonwebtoken) + bcryptjs
  Paiement : Stripe (mode test + webhooks)
  Email    : Nodemailer (SMTP)
  Sécurité : Helmet, CORS, express-rate-limit
  Frontend : HTML5 / CSS3 / JavaScript vanilla (responsive, dark theme)

  DÉMARRAGE
  ─────────
  cd website
  npm install
  cp .env.example .env    ← renseigner les clés API
  npm start               ← http://localhost:3000

  Admin par défaut :
    Email    : admin@axionpad.com
    Mot de passe : Admin1234!

  ROUTES API COMPLÈTES
  ────────────────────

  Auth
    POST   /api/auth/register          Inscription
    POST   /api/auth/login             Connexion → JWT
    GET    /api/auth/me                Profil courant
    PUT    /api/auth/me                Modifier profil
    PUT    /api/auth/password          Changer mot de passe

    POST   /api/products/:id/reviews   Soumettre un avis (achat requis)

    GET    /api/cart                   Contenu du panier + total
    POST   /api/cart                   Ajouter un article
    PUT    /api/cart/:id               Modifier quantité

    DELETE /api/cart                   Vider le panier

  Admin (réservé rôle admin)
    GET    /api/admin/stats            KPIs dashboard
    GET    /api/admin/orders           Toutes les commandes (filtres, pagination)
    GET    /api/admin/orders/:id       Détail commande
    PUT    /api/admin/orders/:id/status    Changer statut

    GET    /api/admin/products         Liste produits + stock
    POST   /api/admin/products         Créer produit
    PUT    /api/admin/products/:id     Modifier produit

    GET    /api/admin/coupons          Liste codes promo
    POST   /api/admin/coupons          Créer code promo

  LOGIQUE MÉTIER
  ──────────────
  Numéro de commande : AXP-YYYYMMDD-XXXX (ex: AXP-20260406-0001)

  Frais de livraison :
    Standard  5,99 €  (< 100 €)
    Express  12,99 €  (< 100 €)
    Gratuit   0,00 €  (≥ 100 €)

  Statuts commande : pending → confirmed → shipped → delivered
                               ↓              ↓
                           cancelled      (irréversible)

  Panier invité : identifié par X-Session-Id (localStorage)
  Panier connecté : lié au user_id
  Fusion au login : POST /api/cart/merge

================================================================================
  NOTES DE SÉCURITÉ (pour mise en production)
================================================================================

  OBLIGATOIRE avant déploiement public :

  1. Changer JWT_SECRET dans .env (clé longue et aléatoire)
  2. Configurer les vraies clés Stripe (sk_live_...)
  3. Configurer SMTP pour les emails de confirmation
  4. Activer HTTPS (certificat SSL via Let's Encrypt ou Cloudflare)
  5. Configurer les variables NODE_ENV=production
  6. Sauvegarder régulièrement axionpad.db
  7. Mettre à jour le mot de passe admin par défaut
  8. Configurer le webhook Stripe avec STRIPE_WEBHOOK_SECRET
